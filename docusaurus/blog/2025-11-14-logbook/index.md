- Added a **chatbot directly integrated into the homepage orb**
- Makes Sofia more interactive and conversational from the first interaction
- Enhances user onboarding and engagement

### ENS Support

- Sofia now **retrieves ENS data** for connected wallets
- Displays **profile pictures and names** linked to users' wallets
- Creates a more personalized and recognizable user experience

- Favicon now **adapts to browser theme** (light/dark mode)
- Improves visual consistency across different user preferences
