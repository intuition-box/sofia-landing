# Pulse Analysis

**Pulse Analysis** lets you analyze all your currently open browser tabs and extract potential signals to certify.

## How It Works

<img src={require('./img/pulse-how-it-works.png').default} alt="Pulse How It Works" style={{maxWidth: '500px', width: '100%'}} />

## What You Can Do

| Action | Description |
|--------|-------------|
| **Launch analysis** | AI analyzes all your open tabs |
| **View signals** | See extracted [triples](../core-concepts/triples.md) by session |
| **Select signals** | Choose individually or in bulk |
| **Amplify signals** | Publish selected signals on-chain |
| **Delete signals** | Remove irrelevant signals or entire sessions |
