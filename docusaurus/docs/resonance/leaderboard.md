# Streak Leaderboard

Compete with other users on daily activity.

## Leaderboards

| Leaderboard | Metric |
|-------------|--------|
| **[Certification](../features/certifications.md) Streak** | Consecutive days with [certifications](../features/certifications.md) |
| **Vote Streak** | Consecutive days with votes |

## Features

- **See rankings**: Compare your position with others
- **Visit profiles**: Check out top users
- **Track progress**: Monitor your streak status
