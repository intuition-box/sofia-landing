# Predicates

Predicates are the "verbs" that connect you to the pages you certify. Each intention type has a corresponding predicate. Learn more about how intentions work in the [Intentions](../features/intentions.md) guide.

## Intention-Based Predicates

| Intention | Predicate |
|-----------|-----------|
| **Work** | "visits for work" |
| **Learning** | "visits for learning" |
| **Fun** | "visits for fun" |
| **Inspiration** | "visits for inspiration" |
| **Buying** | "visits for buying" |
| **Music** | "visits for music" |
| **Trusted** | "trusts" |
| **Distrusted** | "distrusts" |

:::info Coming Soon
The core team is actively working on implementing new predicates to expand the range of relationships you can express with your certifications.
:::
