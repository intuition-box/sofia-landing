November 14, 2025

Logbook 14/11

First of all, we apologize for being quiet over the past two weeks — we've been working hard on deploying Sofia on Phala, migrating to Intuition's mainnet, and setting up the GaiaNet node hosting. During this time, we've made significant progress across infrastructure, features, and user experience. Sofia is now running on mainnet with secure hosting, and we're testing with real users to prepare for broader Alpha release.

Infrastructure & Deployment

  • Migrated to Intuition mainnet for production-level trust operations
  • Deployed on Docker Hub and Phala for secure, decentralized hosting
  • Obtained Phala trust certificate:

    • ➡️ View Certificate

We're now running live tests with real users to measure server consumption and performance.

New Features

Integrated Chatbot in the Orb

  • Added a chatbot directly integrated into the homepage orb
  • Makes Sofia more interactive and conversational from the first interaction
  • Enhances user onboarding and engagement

ENS Support

  • Sofia now retrieves ENS data for connected wallets
  • Displays profile pictures and names linked to users' wallets
  • Creates a more personalized and recognizable user experience

Dynamic Favicon

  • Favicon now adapts to browser theme (light/dark mode)
  • Improves visual consistency across different user preferences

UI / UX Improvements

  • Fixed several UI bugs for better consistency
  • Improved the Account page with refined interactions and layout
  • Enhanced overall user experience across the extension

Backend & Architecture

WebSocket Refactor

  • Complete refactor of the WebSocket system
  • Implemented multi-user management via unique channels
  • Improved real-time communication and scalability

Backend Improvements

  • Substantial backend enhancements for performance
  • Better resource management and optimization
  • Preparing for increased user load

What's Next

  • Analyzing performance metrics from current user tests
  • Preparing for broader Beta testing phase
  • Continuing backend optimization and feature refinements

Sofia is now ready for Alpha testing — stronger infrastructure ready to welcome our Alpha testers and receive their feedback.
